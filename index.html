Política de Privacidad

Esta política de privacidad describe cómo manejamos los datos enviados por los usuarios que interactúan con nuestro sistema automatizado de WhatsApp.

1. Información que recopilamos

Podemos procesar temporalmente la siguiente información:

  • Número de teléfono del usuario
  • Mensajes enviados por el usuario
  • Metadatos asociados a la comunicación

2. Uso de la información

La información recibida se utiliza únicamente para generar respuestas automáticas a mensajes enviados voluntariamente por el usuario.

3. Compartir información

No vendemos, compartimos ni transferimos información personal a terceros.

4. Almacenamiento

No almacenamos datos de manera permanente. Toda la información se procesa de manera temporal únicamente para responder a los mensajes.
